# Poster shop

Återskapa nedanstående layout med hjälp av flexbox och grid. Följande kriterier skall uppfyllas:

* På skärmar upp till 600px skall alla kort visas i en kolumn och vara identiska
* På skärmar mellan 600px och 900px skall de mindre korten visas i 2 kolumner, och de större i 1 kolumn
* På skärmar större än 900px skall de mindre korten visas i 3 kolumner och de större korten i 2 kolumner (samma layout som skissen nedan)


Figmaskissen [hittar du här](https://www.figma.com/file/ApkqGdPVOlTaKVX6r12bGv/Poster-Shop?node-id=0%3A1).
